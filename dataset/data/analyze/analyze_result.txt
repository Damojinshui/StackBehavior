- Android IDE
    Eclipse เป็นหนึ่งใน Integrated Development Environment (IDE) ที่นิยมใช้ในการพัฒนาแอพพลิเคชั่น Android ในยุคก่อนๆ ซึ่งการเปิดตัวของ Android Studio ของ Google ซึ่งเป็น IDE ที่ทำมาเพื่อพัฒนาแอพพลิเคชั่น Android โดยตรงมีผลกระทบต่อ Eclipse ดังนี้
    2013 - Google เปิดตัว Android Studio ทำให้ความสนใจใน Eclipse ลดลงไป ~0.1% จาก ~1.1%
    2014 - ในเดือนธันวาคม Android Studio ได้มีการปล่อยตัวเต็มเวอร์ชั่น 1.0 และ 1.0.1 ให้ใช้งานทำให้ความนิยมเพิ่มขึ้นจาก ~0.09% เป็น ~0.2% ซึ่งความนิยม Eclipse กลับไปอยู่ที่ ~1.01%
    2015 - หลังจากเปิดเวอร์ชั่นเต็ม Android Studio ไปในปลายปีที่แล้วทำให้ความนิยมก็เพิ่มขึ้นอย่างมากจากเดิม ~0.2% เป็น ~0.5% แต่กลับกันในฝั่งของ Eclipse ได้ลดลงอย่างมาก จาก ~1.1% ลงไปเหลือ ~0.7%
    2016 - ในปีนี้ Google ประกาศเลิกสนับสนุนส่วนเสริม Android Developer Tools สำหรับ Eclipse ซึ่งเป็นส่วนหนึ่งที่ใช้ในการพัฒนา ทำให้เป็นครั้งแรกที่ความนิยมของ Android Studio สูงกว่า Eclipse โดยอยู่ที่ ~0.7% และ ~0.5% ตามลำดับ
    และในปีถัดๆมาความนิยมของ Eclipse ก็ลดลงเรื่อยๆจนถึง ~0.4% ในปี 2018 ซึ่งส่วนใหญ่จะเป็นผู้ใช้ที่ใช้ Eclipse ในการพัฒนา Java

พฤติกรรมของผู้ใช้ (Behavior)
- การคอมเม้นใน stack overflow ก็จะมีการคอมเม้นทั้งด้านบวกและด้านลบ 
แต่ส่วนมากก็จะมีคอมเม้นแต่ด้านบวกเป็นส่วนใหญ่เพราะเนื่องจาก stack overflow นั้นเป็นเว็บที่เป็นการตั้งคำถามและ
ตอบปัญหาของเหล่าโปรแกรมเมอร์เลยทำให้ การคอมเม้นที่เป็นด้านลบมีน้อย แต่ไม่ใช่ไม่มี เพราะเมื่อมีคน
มารวมกันมากๆ ทำให้มีหลากหลายความคิดมากขึ้น การคอมเม้นด้านลบอาจจจะเกิดจาก อยากจะคอมเม้นแต่ไม่รู้จะคอมเม้นอะไร 
เห็นว่าเป็นเรื่องง่ายๆทำไมต้องถาม แต่ก็เป็นส่วนน้อยที่มีความคิดแบบนี้ หรืออาจจะเกิดการไม่ชอบเป็นการส่วนตัวกับเจ้าของคำถาม 

กิจกรรมของผู้ใช้ (Activity)
- สำหรับในช่วงต้นเดือนแรกๆ เดือนที่ 1-2 ทีมีปริมาณจำนวนคำถามอยู่ในช่วงกลางๆคาดว่าน่าจะเกิดขึ้นมาจากการที่เพิ่งกลับมาจากช่วงเทศกาลสิ้นปีที่เพิ่งจะหมดไปทำให้การทำงานต่างๆจำนวนคำถามต่างๆอยู่ในช่วงทรงๆตัวยังไม่มีอะไรมากนัก
สำหรับช่วงเดือนที่ 3 ที่มีจำนวนกราฟพุ่งขึ้นมาสูงคาดว่าจะเป็นจากการที่หลังจากผ่านช่วงพักต่างๆมาแล้วอาจจะเป็นช่วงในการเปิดโปรเจคใหม่ๆขึ้นมาทำให้เกิดคำถามมากขึ้นเมื่อที่จะทำโปรเจ็คต่างๆนี้ให้สำเร็จ
สำหรับช่วงเดือน 4-8 คาดอาจจะเป็นช่วงที่ได้รับคำตอบจากคำถามที่ได้ทิ้งเอาไว้ในช่วงเดือนที่ 3 และเดินหน้าทำโปรเจคกันจึงจะมีคำถามเข้ามาอย่างเรื่อยๆแต่จะไม่มากเท่าเดือนที่ 3
สำหรับช่วงเดือนที่ 9-11 เป็นช่วงที่น่าจะเป็นการทำโปรเจคสำเร็จแล้วหรืออยู่ในช่วงที่แก้ไขกันเล็กๆน้อยๆทำให้ไม่ได้เกิดคำถามอะไรกันมากนัก
สำหรับช่วงเดือนที่ 12 เป็นช่วงเดือนพักช่วงสิ้นปีทำให้คนส่วนมากเลือกที่จะหยุดพักผ่อนกันมากกว่าที่จะมาถามคำถาม
